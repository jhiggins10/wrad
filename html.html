HTML tags

HTML (Hyper Text Markup Language) is one of the main buliding blocks of web development. Almost every website uses it

Tags

one of the first things you need to know about in HTML is the use of tags, which are used to help format the document on a web browser. There are multiple types of tags such as :

<head> which sets up the head of the document

<body> which sets up the body of the document


apart from those tags we also have

<i> which sets text to be a italic font

<b> which sets to be in a bold font

<small> which sets the text to be small

and many more

Look at the images above, can you see what makes the text change?

paragraphs and line breaks

a new paragraph will start with the <P> . if you want a line break you can use the <br>
